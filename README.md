# Database Manager

## Objetivo do projeto

O **Database Manager** é uma aplicação desenvolvida em Python com o objetivo de facilitar o gerenciamento de bancos de dados relacionais por meio de uma interface gráfica simples e intuitiva.

O projeto foi criado para praticar conceitos de desenvolvimento desktop, conexão com bancos de dados, organização de código e boas práticas de programação, permitindo realizar operações de gerenciamento de forma prática.

Além de servir como ferramenta de estudo, o projeto demonstra conhecimentos em desenvolvimento de aplicações utilizando Python.

---

## Tecnologias utilizadas

* Python 3
* SQLite
* Git
* GitHub

## Como executar localmente

### 1. Clone o repositório

```bash
git clone https://github.com/CrossGuild33/Database_Manager.git
```

### 2. Acesse a pasta do projeto

```bash
cd Database_Manager
```

### 3. (Opcional, mas recomendado) Crie um ambiente virtual

**Linux/macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows**

```powershell
python -m venv venv
venv\Scripts\activate
```

### 4. Instale as dependências

```bash
pip install -r requirements.txt
```

### 5. Execute a aplicação

```bash
python app.py
```
